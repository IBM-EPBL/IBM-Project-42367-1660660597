How do you use JAMP?

Please refer the below steps to use JAMP's tools. Click to Learn More

1. If you are a registered user, log-in to the system. Else, register using your mail ID.
2. Select the type of disease for which you want to generate risks
3. Input the patient's details.
4. Click Submit button.
5. Get the results based on inputted data(you can download your results as pdf).
6. Decide on your next step based on predicted results.
